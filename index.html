Cryptomite

Cryptomite is a Python library of randomness extractors, created by Quantinuum’s Quantum Cryptography team. At a high level, the library offers state-of-the-art randomness extractors that are easy to use, optimized and numerically precise – providing a trade-off of features that suits numerous practical use cases today.

The performance critical parts of the library (e.g. NTT) are implemented in C++, but the rest of the library (e.g. parameter estimation) is implemented in Python for accessibility and ease of installation.

The package is available for Python 3.8 and higher on Mac, Windows and Linux. To install, type:

pip install cryptomite

To see the example notebooks, go to Examples.
